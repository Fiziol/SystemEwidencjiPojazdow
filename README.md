# SystemEwidencjiPojazdow
# System Zarządzania Flotą i Przeglądami

Projekt semestralny realizujący funkcjonalność ewidencji pojazdów mechanicznych oraz monitorowania terminów badań technicznych.

## Etapy Projektu:
* **Etap 1:** Projekt interfejsu użytkownika i podstawowa logika klas.
* **Etap 2:** Implementacja modułu raportowania (RDLC) i zestawów danych (DataSet).
* **Etap 3:** Obsługa zapisu i odczytu danych z plików lokalnych.
* **Etap 4:** Optymalizacja kodu, obsługa wyjątków i finalne testy aplikacji.

## Technologie:
* C# / .NET Framework
* Windows Forms
* Microsoft Reporting Services
